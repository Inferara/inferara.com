How it works 
        flowchart LR
            Inference --> WAT
            WAT --> WASM
            WASM --> V;

Inference compilation process consists of the following steps:

  • Transpile Inference code to WebAssembly Text (WAT)
  • Compile WAT to WebAssembly (WASM)
  • Translate WASM to V (proof assistant code)
